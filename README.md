# PokedexAndroid

Aplicación Android desarrollada en **Java** que funciona como una Pokédex: permite consultar información de Pokémon de forma rápida y visual.

## 📱 Descripción

**PokedexAndroid** es una app móvil para explorar Pokémon, ver sus datos principales y navegar por su información detallada desde una interfaz sencilla.

## ✨ Funcionalidades

- Listado de Pokémon.
- Vista de detalle por Pokémon.
- Visualización de imagen, nombre y datos básicos.
- (Opcional) Búsqueda por nombre.
- (Opcional) Carga de datos desde API (por ejemplo, PokéAPI).
- (Opcional) Manejo de errores y estados de carga.

## 🛠️ Tecnologías

- **Lenguaje:** Java
- **Plataforma:** Android
- **IDE recomendado:** Android Studio
- **Build system:** Gradle

## 🧱 Estructura del proyecto 

- `app/`
  - `src/main/java/...` → lógica de la app (Activities, Fragments, adapters, etc.)
  - `src/main/res/` → layouts, strings, drawables, temas
  - `AndroidManifest.xml`

## 🚀 Instalación y ejecución

1. Clona el repositorio:
   ```bash
   git clone https://github.com/camachoo18/PokedexAndroid.git
   ```
2. Abre el proyecto en **Android Studio**.
3. Espera a que Gradle sincronice dependencias.
4. Ejecuta la app en un emulador o dispositivo físico.

## ⚙️ Requisitos

- Android Studio (versión reciente)
- JDK compatible con tu versión de Android Studio
- SDK de Android instalado
